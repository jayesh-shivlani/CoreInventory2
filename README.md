# Core Inventory 2

Unified workspace for the Core Inventory frontend and backend.

## Prerequisites

- Node.js 18+
- npm 9+

## One-time setup

```bash
npm install
npm run install:all
```

## Run full stack

```bash
npm run dev
```

Services:

- Frontend: http://localhost:5173
- Backend API: http://localhost:4000/api
- SQLite DB: backend/data/coreinventory.db
